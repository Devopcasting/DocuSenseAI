# DocuSense AI

DocuSense AI is an intelligent document query tool designed to enable users to ask questions across diverse document formats including PDFs, DOCX, images, Excel files, and databases. This tool leverages FastAPI to provide precise and easy information retrieval, simplifying document exploration and analysis.

## Key Features

- **Multi-format Support**: Query across various document formats seamlessly.
- **Natural Language Processing (NLP)**: Use natural language to ask questions about your documents.
- **FastAPI Backend**: Utilizes FastAPI for efficient and scalable performance.
- **Document Exploration**: Quickly navigate through documents to extract insights.
- **Interactive Query Interface**: User-friendly interface to interact with your documents.
